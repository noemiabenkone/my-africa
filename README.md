
My Africa 🌍

Projeto web para explorar países africanos e conversar com uma IA sobre cada país.

O usuário clica em um país e pode perguntar sobre cultura, língua, história e informações gerais, recebendo respostas
geradas por IA.

Tecnologias

Next.js

React + TypeScript

Tailwind CSS

REST Countries API

GEMINI_API

Aprendizados

Integração segura com IA no backend

Uso de variáveis de ambiente

Organização e refatoração de código

Rodar o projeto

1. Clone o repositório

2. Instale as dependências:

   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente:

   - Crie um arquivo `.env.local` na raiz do projeto.
   - Adicione sua chave do Google Gemini: `GEMINI_API_KEY=sua_chave_aqui`
   - Você pode obter uma chave gratuita no [Google AI Studio](https://aistudio.google.com/).
   
4. Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```
